## InputBindingInteractionType

`enum` in `AdvancedSceneManager.Models`  /  Inherits from: `Enum`### Description
Specifies the interaction type to use for scene bindings.

### Static Fields

| Member | Description |
|--------|-------------|
| `InputBindingInteractionType Hold` | Specifies that the scene or collection will be opened automatically on button down, then closed on button up. |
| `InputBindingInteractionType Open` | Specifies that the scene or collection will be opened automatically, but not closed. |
| `InputBindingInteractionType Toggle` | Specifies that the scene or collection will be opened automatically on button down, then closed on next button down. |
