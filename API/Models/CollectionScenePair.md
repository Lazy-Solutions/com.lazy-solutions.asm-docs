## CollectionScenePair

`class` in `AdvancedSceneManager.Models`### Description
Represents a pair of a scene collection and an index, used to reference a specific scene relative to its parent collection.

| Member | Description |
|--------|-------------|
| `ISceneCollection collection` | Gets or sets the scene collection. |
| `Scene scene` | Gets or sets the scene. |

### Fields

| Member | Description |
|--------|-------------|
| `int32 sceneIndex` | Gets or sets the index of the scene within its collection. |
