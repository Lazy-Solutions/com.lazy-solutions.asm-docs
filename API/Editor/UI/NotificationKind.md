## NotificationKind

`enum` in `AdvancedSceneManager.Editor.UI`  /  Inherits from: `Enum`### Description
Specifies the kind of notification.

<b> Remarks:</b>
Only available in the editor.

### Static Fields

| Member | Description |
|--------|-------------|
| `NotificationKind Default` | Does not indicate anything in particular for the notification. Displays no icon. |
| `NotificationKind FixUp` | Indicates that the notification is meant to convey a warning. Displays a warning icon. |
| `NotificationKind Info` | Indicates that the notification is meant to convey information. Displays an info icon. |
| `NotificationKind Link` | Indicates that the notification is meant to act as a link to something. Displays a link icon. |
| `NotificationKind Profile` | Indicates that the notification is meant to convey something relating to profiles. Displays a profile icon. |
| `NotificationKind Scene` | Indicates that the notification is meant to convey something relating to scenes. Displays a scene icon. |
